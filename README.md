# Coffee House Café Website

## Description
Coffee House Café is a modern and responsive website designed for a coffee shop. The website showcases the café's menu, services, gallery, and contact information in a clean and user-friendly interface.

## Features
- Responsive Design
- Modern UI/UX
- Coffee Menu Section
- About Us Page
- Image Gallery
- Contact Form
- Google Maps Integration
- Mobile Friendly
- Smooth Animations

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Pages
1. Home
2. Menu
3. About Us
4. Gallery
5. Contact

## Purpose
The purpose of this project is to provide a professional online presence for a coffee shop and improve customer engagement.
